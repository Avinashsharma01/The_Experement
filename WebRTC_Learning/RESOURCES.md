## Glossary

- WebRTC: browser APIs and protocols for real-time communication.
- SDP: text format describing media and transport negotiation.
- ICE: framework to discover and select best network path.
- STUN: helps discover public address candidates.
- TURN: relay server when direct peer path is not possible.
- DTLS: encryption handshake for secure channels.
- SRTP: encrypted media transport.
- Data Channel: peer-to-peer arbitrary data transport.
- SFU: server forwards participant streams without mixing.
- MCU: server mixes streams into fewer outputs.
