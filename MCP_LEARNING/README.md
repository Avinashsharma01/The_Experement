# 🎓 MCP Servers in VS Code — Complete Course

> **Learn how to supercharge VS Code with MCP (Model Context Protocol) servers — from zero to hero!**

---

## 📋 What You'll Learn

By the end of this course, you will:
- ✅ Understand what MCP is and why it matters
- ✅ Set up and configure MCP servers in VS Code
- ✅ Use the **GitHub MCP Server** to manage repos, issues, and PRs with AI
- ✅ Use the **Filesystem MCP Server** to let AI read/write files on your machine
- ✅ Use the **Fetch MCP Server** to let AI browse the web
- ✅ Use **Database MCP Servers** (SQLite, PostgreSQL)
- ✅ Discover and install community MCP servers
- ✅ Build confidence to add ANY MCP server to your workflow

## 🚀 How to Use This Course

1. **Read modules in order** — each one builds on the previous
2. **Try every example** — don't just read, actually configure and test
3. **Use the cheat sheet** as a quick reference while working
4. **Start with Module 1** if you're brand new, or jump to Module 3 if you already know the basics

---

## 🔧 Prerequisites

- **VS Code** version 1.101 or later (for remote MCP & OAuth support)
- **GitHub Copilot** extension installed and active
- **Docker** installed (optional, for some local servers)
- **Node.js** installed (for some community servers)
- A **GitHub account** (for the GitHub MCP Server module)

---

> 💡 **Tip:** MCP is evolving fast! This course covers the core concepts that stay the same, even as new servers appear.
