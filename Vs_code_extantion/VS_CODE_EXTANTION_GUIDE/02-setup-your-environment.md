# 2) Setup Your Environment

## Prerequisites

Install these tools first:

- **Node.js LTS** (includes `npm`)
- **Visual Studio Code**
- **Git** (optional, but recommended)

## Install scaffolding tools

Open terminal and run:

```bash
npm install -g yo generator-code
```

## Verify installation

```bash
node -v
npm -v
yo --version
```

If these commands return versions, your setup is ready.

## Choose TypeScript or JavaScript?

- Pick **TypeScript** if you want stronger typing and better maintainability.
- Pick **JavaScript** if you want the fastest start.

For learning and future growth, TypeScript is usually the better choice.

## Useful optional tools

```bash
npm install -g vsce
```

`vsce` is used later to package and publish your extension.
